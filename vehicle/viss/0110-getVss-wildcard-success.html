[0110: get a VSS tree with a valid request with a wildcard]

Test assertion:
When the client makes a 'getVSS' request to the server
with a data path contains a wildcard, 'vssSuccessResponse' is returned
which contains VSS tree matches to the wildcard data path.

Test Scenario:
1. Execute getVSS method without specifing a path
2. Confirm 'vssSuccessResponse' will be returned and it contains entire VSS tree.

Expected Result:
- 'VssSuccessResponse' is retruned and it contains VSS tree matches to wildcard data path.
